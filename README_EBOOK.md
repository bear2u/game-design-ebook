# 게임 디자인의 기술 - 이북 사이트

번역된 챕터들을 읽을 수 있는 간단한 웹 이북 사이트입니다.

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다.
2. 왼쪽 사이드바의 목차에서 원하는 챕터를 클릭합니다.
3. 챕터 내용이 오른쪽에 표시됩니다.

## 기능

- 📚 25개 챕터 목차
- 📖 클릭 한 번으로 챕터 읽기
- 📱 반응형 디자인 (모바일 지원)
- 🎨 깔끔한 블로그 스타일 UI
- 🔍 빠른 챕터 탐색

## 파일 구조

```
game-design-translate/
├── index.html          # 메인 이북 사이트
├── translate/          # 번역된 챕터 파일들
│   ├── chapter1.md
│   ├── chapter2.md
│   └── ...
└── imgs/              # 이미지 파일들 (있는 경우)
```

## 로컬 서버 실행 (선택사항)

CORS 문제를 피하기 위해 로컬 서버를 실행하는 것을 권장합니다:

### Python 3
```bash
python -m http.server 8000
```

### Node.js (http-server)
```bash
npx http-server -p 8000
```

그 다음 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 브라우저 지원

- Chrome (권장)
- Firefox
- Safari
- Edge

## 참고사항

- 이미지 파일(`imgs/` 폴더)이 있는 경우 정상적으로 표시됩니다.
- 이미지가 없는 경우에도 텍스트는 정상적으로 표시됩니다.
- 마크다운 파일은 클라이언트 사이드에서 HTML로 변환됩니다.
