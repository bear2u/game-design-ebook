# Interactive Ebook Template Skill

A reusable template for creating beautiful, mobile-optimized ebook websites.

## Quick Start

1. Copy `assets/` folder contents to your project
2. Add your chapters to `translate/` folder
3. Update chapter list in `index.html`
4. Deploy to GitHub Pages

## Documentation

- **SKILL.md**: Main skill documentation
- **references/EXAMPLES.md**: Usage examples
- **references/DEPLOYMENT.md**: Deployment guide
- **references/CUSTOMIZATION.md**: Customization options

## Features

✅ 3 themes (Light, Dark, Warm)  
✅ Font size control  
✅ Mobile optimized  
✅ Chapter navigation  
✅ Keyboard shortcuts  
✅ GitHub Pages ready
