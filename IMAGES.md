# 이미지 파일 안내

## ⚠️ 중요: 이미지 폴더 필요

이 이북 사이트는 이미지를 표시하기 위해 `imgs/` 폴더가 필요합니다.

## 이미지 파일 위치

원본 `book.md` 파일에서 이미지 경로는 다음과 같습니다:
```
imgs/img_in_image_box_*.jpg
```

## 해결 방법

### 옵션 1: 이미지 폴더 생성 및 이미지 추가

1. 프로젝트 루트에 `imgs` 폴더 생성:
   ```bash
   mkdir imgs
   ```

2. 원본 PDF나 소스에서 이미지 파일들을 `imgs/` 폴더에 복사

3. 이미지 파일명 형식: `img_in_image_box_*.jpg`

### 옵션 2: 이미지 없이 사용

이미지가 없어도 텍스트는 정상적으로 표시됩니다. 이미지 부분만 표시되지 않습니다.

## 현재 상태

- ✅ 이미지 경로 처리 로직: 완료
- ⚠️ `imgs/` 폴더: 없음 (생성 필요)
- ⚠️ 이미지 파일: 없음 (추가 필요)

## GitHub Pages 배포 시

이미지 파일들을 `imgs/` 폴더에 추가한 후 GitHub에 푸시하면 이미지가 정상적으로 표시됩니다.
